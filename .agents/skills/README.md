# Project skills

The `huskarui` and `qmlpreviewer` skills are vendored from PyHuskarUI commit
`4e2c8f156b8a9dc2bf67f4e2e06f7f124963b370`.

Update them by copying `agent/skills/huskarui` and `agent/skills/qmlpreviewer` from a reviewed
PyHuskarUI revision. Review the diff, update the commit above, and verify the metadata query and
QML preview scripts before committing.
